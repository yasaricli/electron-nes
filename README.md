# Nes Emulator
Electron Nes emulator.

## Quick Start

```bash
$ git clone https://github.com/yasaricli/electron-nes.git
$ cd electron-nes
$ npm install
$ npm start
```

## Watch less styles
```bash
$ gulp watch
```

## Build app in OSX
```bash
$ electron-packager . --platform=darwin --arch=x64 --version=0.36.9 --icon=assets/img/icon.icns
```
